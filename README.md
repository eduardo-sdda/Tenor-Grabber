# Tenor Grabber

Download every GIF uploaded by any Tenor user and bundle them into a ZIP file.

Runs **100% in your browser** — no servers, no accounts, no data collection.

🔗 Live tool: https://eduardo-sdda.github.io/Tenor-Grabber/

---

## Features

- Download **all GIFs from a Tenor user**
- Packs everything into a **single ZIP**
- Runs **entirely in the browser**
- **No backend**
- **No login required**
- Works on desktop and mobile

---

## How to use

1. Open the website  
2. Enter a Tenor username  
3. Click **GRAB**  
4. Download the generated ZIP file

---

## How it works

The tool uses the Tenor API to fetch all GIFs from a user and downloads them directly in the browser.  
The files are then packed using **JSZip** before being downloaded as a ZIP archive.
